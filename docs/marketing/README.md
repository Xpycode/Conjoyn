# Conjoyn Marketing — Playbook Index

> All marketing docs in one place, in suggested execution order. Each file is self-contained; this is
> the map. Core thread across every doc: **lead with the outcome, not the tech; the SRT-telemetry +
> auto-grouping (and TC/metadata correctness for pros) is the moat; intercept buyers at the moment of
> intent; help first, sell second.**

## The two audiences (keep straight everywhere)
- **Hobbyist / prosumer** → hook = the **split-file merge** ("one flight, five files"). High volume.
  Reached via forums + SEO. Lossless + drag-and-drop is the pitch.
- **Pro / cinematographer** → hook = **timecode + metadata correctness** + SRT sync. Higher willingness
  to pay. Reached via targeted creator outreach (e.g. Bloom). "Your TC comes out *right*" is the pitch.

## The one rule
Don't parallelize all of this. Sequence it. One acquisition wedge done well (community answering + the
cornerstone tutorial) beats five half-built channels. Every two weeks, kill the lowest-ROI activity and
reinvest the hours into the best one.
