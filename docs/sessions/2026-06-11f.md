## Progress

### Wave 1 — Bug fixes (4 tasks, parallel agents)

**Task 1 · "Processing…" label for single-file jobs**
`statusText` in `QueueRow` returned "Joining…" for all active jobs regardless of clip
count. Added `clips.count == 1 ? "Processing…" : "Joining…"` branch.

**Task 2 · Chip filter: suppress `.info` severity**
`verificationSection` filtered `severity > .pass`, catching `.info` (the normal ≤1-frame
duration delta from `-fflags +genpts`). Changed to `>= .warning`. No more false ⚠
Duration chip on every clean join.

**Task 3 · mapStatus: thorough hash pass → `.verified`**
When Tier 1 flagged a warning and Tier 2 hash passed, `mapStatus` still returned
`.warning` (orange seal) because `result.passed` uses worst-wins. Added a leading guard:
if `result.tier == .thorough` and a `.pass` hashMatch check exists → return `.verified`
immediately. Orange seal no longer sticks after byte-exact confirmation.

**Task 4 · Seal animation: immediate stop**
`VerificationSeal` used `.animation(.default, value: spin)` on the view, causing a spring
snap-back when spin was set to false. Replaced with explicit `withAnimation(nil) { spin
= false }` in `.onChange` handler. Rotation stops cleanly on terminal state.

+2 tests → 317/317. Wave 1 merged to `main` (`71d58c2`), pushed.

### Wave 2 — New UI features (single agent, both touch QueuePanel.swift)

**Task 5 · Queue restore banner**
`queue.json` reloaded silently. Added `@Published var restoredJobCount: Int` to
`QueueManager`, set in `loadQueue()` (count of restored pending jobs). Added
`RestoreBanner` sub-view in `QueueSection`: shows "Restored N job(s) from last session"
with Dismiss (session-local) and Clear Pending (`clearPendingJobs()` new method on
`QueueManager+Operations.swift`).

**Task 6 · Queue row: job type + duration + size**
`liveMetrics` was blank for non-active rows. Added `else if !isActive` branch: shows
`SINGLE · 5:14 · 1.4 GB` or `4 files · 51:39 · 13.7 GB` using `CJFormat.duration` +
`CJFormat.size`. Added `totalSourceBytes: Int64` computed property to `ConversionJob`
(`clips.reduce(0) { $0 + $1.totalFileSize }`). 317/317. Wave 2 merged to `main`
(`2f9455e`), pushed.
