## The script — `01_Project/scripts/make-dmg.sh`
- Mirrors `notarize.sh` conventions (config block, `bold()`, `set -euo pipefail`, inside-out flow).
- **DRY:** delegates the app build + sign + verify + notarize + staple to `notarize.sh` (single
  source of truth for app signing), then handles only the DMG.
- DMG steps: `create-dmg` (volume icon from the app's `AppIcon.icns`, 600×400 window, app icon +
  `/Applications` drop-link, hidden extension) → `codesign --timestamp` (Developer ID Application)
  → `notarytool submit --wait` → `stapler staple` → `spctl -t open` assessment.
- Robustness: preflight checks (create-dmg installed, notary profile exists); one retry on
  transient `hdiutil resource busy`; `SKIP_APP=1` reuses an already-stapled app to skip a second
  app-notarization round-trip when iterating on DMG layout.
- Installed `create-dmg` 1.2.3 via Homebrew.
