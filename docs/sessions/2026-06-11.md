### Verification / diagnosis
- **SRT path live-confirmed** on the SRT-bearing card (Jun 7 2026 footage): every row resolved

- **Thumbnail "not loading" is not a bug:** `ThumbnailManager` bounds extraction to **3 concurrent

## Decisions
- Recordings-row integrity = **date-provenance story**, not source-`tmcd` comparison (DJI has no
  `tmcd`). Reuse the engine resolver so the row can never contradict what gets stamped.
- Integrity chip strip is **problem-only**; the routine "date from X" lives as the inline date-line
  tag, never as a chip (no duplication).
- Hide the progress bar for queued (pending) jobs; thumbnail placeholder is a bare flat tile.
